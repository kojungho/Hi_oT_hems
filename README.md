# 힐스테이트 HEMS (Hi-oT HEMS) for Home Assistant

현대오토에버의 **하이오티(Hi-oT) 플랫폼** API를 연동하여, 아파트 월패드의 실시간 및 통계 검침 데이터를 홈어시스턴트(Home Assistant)로 수집하는 커스텀 컴포넌트입니다.

---

##  주요 기능
* **5대 에너지 모니터링**: 전기, 가스, 수도, 온수, 난방 데이터 지원
* **통계 센서 자동 생성**: 당월 누적 사용량, 동일평수 평균, 전년 동월 사용량, 전년 대비 증감량 연산 센서 제공
* **에너지 대시보드 완벽 호환**: 누적 사용량 센서에 `TOTAL_INCREASING` 속성이 부여되어 HA 에너지 대시보드에 즉시 등록 가능

---

## 🛠 HACS 설치 방법

1. Home Assistant에서 **HACS** 메뉴로 이동합니다.
2. 우측 상단 점 3개 메뉴를 누르고 **커스텀 저장소 (Custom Repositories)**를 선택합니다.
3. **저장소 URL**에 본인의 저장소 주소를 입력합니다:
   `https://github.com/kojungho/Hi_oT_hems`
4. **범주 (Category)**에서 **통합구성요소 (Integration)**를 선택한 뒤 **추가 (Add)**를 누릅니다.
5. 목록에 나타난 **힐스테이트 HEMS**를 클릭하고 우측 하단의 **다운로드**를 누릅니다.
6. Home Assistant를 **재시작**합니다.

---

## ⚙️ 설정 방법

1. HA 설정 ➡️ 기기 및 서비스 ➡️ **통합구성요소 추가** 클릭
2. **힐스테이트 HEMS** 검색 후 선택
3. 하이오티(Hi-oT) 앱 계정 정보(ID/PW) 및 데이터 갱신 주기(분 단위, 기본 60분) 입력 후 제출
